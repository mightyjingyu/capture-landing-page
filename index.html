김치찜

캡쳐를 정리하고, 필요한 순간 바로 찾아주는 앱

앱 소개

김치찜은 사용자가 저장한 캡쳐 이미지를 자동으로 분류하고, 일정/학습/재미/증빙 등 다양한 카테고리로 정리해줍니다. 특히 의류/제품 카테고리에서는 쿠팡 파트너스 API와 연동하여 관련 상품을 바로 확인할 수 있는 링크를 제공합니다.

주요 기능

  • 캡쳐 이미지 자동 분류
  • 기한 임박 알림 기능
  • 의류/제품 → 쿠팡 상품 링크 제공
  • 사용자 친화적인 갤러리형 인터페이스

현재 상태

김치찜은 현재 개발 중이며, iOS/Android 앱으로 곧 출시될 예정입니다. 이 페이지는 쿠팡 파트너스 매체 등록을 위한 공식 안내 페이지입니다.
